How to Format BsDatePicker to TimeDate in Angular 9

March 25, 2020 2 minutes read
Angular • TypeScript • JavaScript

Introduction

In these tutorial I will discuss on how to add datepicker and convert to dateTime/any formart in our Angular Forms.

Getting Started

BsDatepicker is a package found in ngx-bootstrap which contains all core (and not only) Bootstrap components powered by Angular.

Installation instructions

Install ngx-bootstrap from npm:

npm install ngx-bootstrap --sav

Note: Here, don’t forget to import FormsModule in main module before start using the input elements.

To add needed package to NgModule imports:

import { BrowserAnimationsModule } from '@angular/platform-browser/animations';
import { BsDatepickerModule } from 'ngx-bootstrap/datepicker';

@NgModule({
imports: [
    BrowserAnimationsModule,
    BsDatepickerModule.forRoot(),
    ...]
})

Note: If you installed ngx-bootstrap not via ng add command, you will need to perform a several actions

Notable change is additional css for it “/datepicker/bs-datepicker.css”

There are two ways of adding css:

Load it from CDN. Add

to your index.html Load it from angular.json file by adding it to style array, just as shown in the image

node_modules/ngx-bootstrap/datepicker/bs-datepicker.css

Next, let’s see how the datepicker can be called in the HTML template. Then go ahead and add bsDatepicker to your template input element

<div class="row">
<div class="col-xs-12 col-12 col-md-4 form-group">
    <input type="text"
        placeholder="Datepicker"
        class="form-control"
        bsDatepicker>
</div>
</div>

if everything is correct you should see the datepicker once you click in the input box.

I wont go deep on how to format the BsDatepicker such as to change theme and disable some defaults settings like week, months and so on. For information on how to do that follow this link Go to https://valor-software.com/ngx-bootstrap/#/datepicker#inline-datepicker

Formating to DateTime

To be able to format date to a format that can be sent to the endpoint we are going to deconstruct the value received from the form, so we can get the various date field needed in your conponents.ts

OnSubmit(form: FormGroup){
        const {value, valid} = form
        if(valid){   
            var date = value.date_of_earning
            var year = date.getFullYear()
            var month = date.getMonth() +1;
            var day = date.getDate()
            var hours = date.getHours()
            var minutes = date.getMinutes()
            var seconds = date.getSeconds()
            // we are setting the new value for the date form constrol
            value["dateTime"] = year+"-"+month+"-"+day +" " + hours+":"+minutes+":"+seconds
            console.log(value)
        }
    }
